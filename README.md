# Let Me Drive 🚗

Let Me Drive is a web-based car rental management system that allows users to browse available cars and make bookings through a simple web interface.

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Java Servlets
- JDBC

Database
- PostgreSQL

Build Tool
- Gradle

Server
- Apache Tomcat

## Features

- User registration and authentication
- Browse available cars
- Car booking functionality
- Booking management
- Database integration with PostgreSQL

## Project Structure

src/main/java  
├── controller  
├── model  
├── repository  
├── servlet  
└── util  

database  
└── schema.sql  

## Getting Started

1. Clone the repository

git clone https://github.com/yourusername/let-me-drive.git

2. Setup PostgreSQL database

3. Configure database connection in the project

4. Run the project using Apache Tomcat
